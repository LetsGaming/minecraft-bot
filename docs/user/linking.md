# Account Linking

Some commands require your Discord account to be linked to your Minecraft account. Linking is a one-time process.

## How to Link

1. In Discord, type `/link`. You'll receive a 6-character code (only visible to you).
2. Join the Minecraft server and type `!link CODE` in chat, replacing `CODE` with the code you received.
3. That's it — your accounts are now linked.

The code expires after 5 minutes. If it expires, just run `/link` again.

## Check Your Link Status

Use `/linkstatus` to see which Minecraft account is linked to your Discord.

## Unlinking

Use `/unlink` to remove the connection. You can re-link at any time with `/link`.

## What Requires Linking?

- `/daily` — claiming daily rewards
- `/netherportal` — needs to read your in-game position
- `/chunkbase` — optionally centers the map on your position
- `/playerhead` — giving yourself a player head in-game
- `/streak` — viewing your daily streak

If you try one of these commands without linking, you'll get an error message reminding you to link first.
